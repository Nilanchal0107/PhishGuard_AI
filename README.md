# PhishGuard AI

Minimal full-stack demo for phishing detection in Indian-language text and audio, powered by Gemini 1.5 Flash.

## Setup

1. Put your API key in `phishguard/.env`:

```
GEMINI_API_KEY=YOUR_KEY_HERE
```

2. Start backend:

```
cd server
npm install
node index.js
```

3. Start frontend:

```
cd client
npm install
npm run dev
```
